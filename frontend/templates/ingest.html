How to Use
  1. Get Google Drive URL:
    • Go to Google Drive
    • Right-click on a folder or file
    • Select "Get link"
    • Copy the URL
  2. Paste URL: Enter the copied URL in the field above
  3. Start Ingestion: Click "Start Ingestion" to process documents
  4. Query Documents: Once processed, you can ask questions about the content
Supported Document Types
  • PDF Documents
  • Word Documents (.docx)
  • Text Files (.txt)
  • Excel Files (.xlsx)
  • PowerPoint (.pptx)
  • Markdown (.md)
